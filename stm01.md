さらにイベントを追加する
```uml
@startuml
[*] --> トップページ

トップページ--> トップページ : 「商品一覧」「商品検索」をクリック
トップページ--> カート内処理 : 「カートの中」クリック
トップページ--> 登録情報 : 登録情報をクリック
トップページ--> 商品詳細 : 楽器画像および名称をクリック


商品詳細--> トップページ : 前の画面に戻るをクリック
商品詳細--> トップページ : 商品一覧をクリック
商品詳細--> トップページ : 商品検索をクリック
商品詳細--> 登録情報 : 登録情報をクリック






登録情報 --> トップページ : 商品一覧をクリック
登録情報 --> トップページ : 商品検索をクリック
登録情報 --> カート内処理 : カートの中をクリック
登録情報 --> 登録情報 : 登録情報をクリック

state ログイン処理{
[*] --> トップページ : 全ページより遷移可能
}

商品詳細--> カート内処理 : カートにいれるをクリック

state カート内処理{
[*] --> カート:
カート --> 注文処理:「注文確定」クリック
カート--> カート: 「カートの中」をクリック
カート --> 商品詳細 : 詳細へをクリック
カート--> [*]: 「商品一覧」「商品検索」クリック
state 注文処理{
[*]--> 注文確定: 
[*]--> 商品詳細 : 「詳細へ」をクリック
[*]--> [*]: 「削除」で削除処理
注文確定--> [*]: 「商品一覧へ戻る」「商品一覧」「商品検索」クリック
}
注文確定 --> カート : カートの中をクリック
注文確定 --> 登録情報 : 登録情報をクリック
}

カート内処理--> 登録情報 : 「登録情報」をクリック


@enduml
```
