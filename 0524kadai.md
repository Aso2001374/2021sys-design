```uml
@startuml
[*] --> トップページ
トップページ -> 会員登録:「会員登録はこちら」をクリック
トップページ --> ユーザーログイン:「ログイン」をクリック
ユーザーログイン--> トップページ: ログイン処理成功
ユーザーログイン--> ユーザーログイン: ログイン処理失敗

トップページ -left-> 弁護士用ページ
state 弁護士用ページ{
[*] --> 弁護士ログイン:「弁護士の方はこちら」をクリック
弁護士ログイン--> 弁護士ログイン: ログイン処理失敗
弁護士ログイン--> 弁護士マイページ: ログイン処理成功
弁護士マイページ--> 弁護士情報変更: 「情報を変更する」をクリック
弁護士情報変更--> 弁護士マイページ: 「変更の確定」をクリック
弁護士マイページ--> [*]: 「トップ」をクリック
}
弁護士用ページ -right-> トップページ

トップページ --> 弁護士検索:「弁護士をさがす」をクリック
弁護士検索 --> 検索結果: 条件を入力した上で「検索」をクリック
検索結果 -down-> 弁護士プロフィール: 各弁護士の写真もしくは名前をクリック
弁護士プロフィール -left-> 口コミ関連:「口コミを見る」をクリック
弁護士プロフィール --> 個人情報取り扱い同意:「この弁護士に依頼」をクリック
個人情報取り扱い同意-->依頼完了:「同意する」をクリック
依頼完了-->トップページ:「トップへ戻る」をクリック

state 口コミ関連{
[*] --> 口コミ一覧 : 「口コミを見る」をクリック
口コミ一覧 --> 口コミ書き込み:「口コミを書く」をクリック
口コミ書き込み --> 書き込み確認:「完了」をクリック
書き込み確認 --> 書き込み完了:「投稿する」をクリック
書き込み完了 --> 口コミ一覧:「口コミ一覧へ戻る」をクリック
口コミ一覧 --> [*]:「トップへ戻る」をクリック
}
口コミ関連 --> トップページ

@enduml
```
